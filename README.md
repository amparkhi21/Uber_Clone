The Uber Clone is a front-end web project that replicates the design and layout of the official Uber website. The goal of this project was to strengthen my understanding of HTML structure, CSS styling, layout techniques, and responsive design.

This project demonstrates my ability to convert a real-world website interface into a working static webpage using pure HTML and CSS.

🚀 Features:-
Responsive landing page design
Navigation bar with menu options
Hero section with call-to-action
Service information sections
Clean and structured layout
Footer with useful links
Mobile-friendly styling using CSS

🛠️ Technologies Used
HTML5 – For webpage structure
CSS3 – For styling and layout
Flexbox – For responsive alignment
CSS Flex – For structured layout design
